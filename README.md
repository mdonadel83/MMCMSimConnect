# MMCM SimConnect ‚Äì SimHub Plugin

MMCM SimConnect is a SimHub plugin designed for **MMCM Racing Championships**.  
It ensures fair play, reliable communication, and real-time monitoring during official events.

## What Does MMCM SimConnect Do?

‚úÖ Verifies that the driver is registered for the race

## How Does It Work? ü§î

### 1. Automatic Driver Identification
When you join an online session, the plugin reads your in-game driver name and checks whether you are registered for the MMCM Racing event.

Just like arriving at a real event and finding your name on the entry list.

---

### 2. Vehicle Data Collection
Every **60 seconds**, the plugin collects telemetry data from your car:

- Fuel level  
- Tire temperatures  
- Vehicle damage  
- Current lap  
- Track name  

It also:
- Manages lap counting  
- Validates fast laps for **Pre-Qualifying**

This is similar to the telemetry systems used by real racing teams.

---

### 3. Fair Play Monitoring
To ensure honest competition, the plugin verifies that:

- No cheat or unauthorized programs are running  
- Fuel consumption values are realistic  

Comparable to the technical inspections performed in real motorsport.

---

### 4. Race Control Voice Messages üîä
When Race Control needs to communicate important information (e.g. **Safety Car procedures** or driver instructions), the plugin:

- Receives the message from the server  
- Reads it aloud in **Italian or English**  
- Automatically lowers the volume of other applications  
- Restores the original audio levels afterward  

Just like a real race radio system.

---

### 5. ESC Key Detection During Races ‚å®Ô∏è
For added realism, the plugin detects when the **ESC key** is pressed during a race.

This helps to:
- Identify voluntary disconnections  
- Analyze incidents or technical issues  

‚ö†Ô∏è No automatic penalties are applied ‚Äî the data is used only for analysis.

---

Imagine a football championship without referees ‚Äî chaos.

**MMCM SimConnect** acts like both the **referee and VAR system**:

- Ensures fair play  
- Enables official race communications

## Installation Guide

### 1. Install SimHub
Download and install SimHub from the official repository:  
üëâ https://github.com/SHWotever/SimHub

---

### 2. Install MMCM SimConnect Plugin
- Download the latest release from the **Releases** section
- Navigate to the `Bin/Release` folder
- Copy the plugin files into your SimHub installation directory

---

### 3. Enable the Plugin
- Start SimHub  
- Enable the **MMCM SimConnect** plugin  
- Enable visibility in the SimHub menu  
- Open the **MMCM SimConnect** menu and log in using your MMCM credentials  

The login remains active for **60 days** and will persist between SimHub restarts.  
After 60 days, revalidation is required.
